Clonar el repositorio
-----------------------------------------------------------------------------------------------------------------------------------
	 git clone https://github.com/obesofi/LaMisionDelAprendiz_web
___________________________________________________________________________________________________________________________________
Actualizar las modificaciones del repositorio
-----------------------------------------------------------------------------------------------------------------------------------
	$ git pull
___________________________________________________________________________________________________________________________________
Revisar el estado de los archivos (Si estan agregados o no)
-----------------------------------------------------------------------------------------------------------------------------------
	$ git status
___________________________________________________________________________________________________________________________________
Agregar los archivos
-----------------------------------------------------------------------------------------------------------------------------------
	$ git add .
___________________________________________________________________________________________________________________________________
Actualizar el repositorio
-----------------------------------------------------------------------------------------------------------------------------------
	$ git push
___________________________________________________________________________________________________________________________________
Agregar un cometario
-----------------------------------------------------------------------------------------------------------------------------------
	$ git commit -m "Comentario"
___________________________________________________________________________________________________________________________________
Actualizar y subir al repositorio
-----------------------------------------------------------------------------------------------------------------------------------
	$ git push
